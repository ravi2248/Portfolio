Emotion Classification from Text

Project Goal

To develop and deploy a robust deep learning model capable of accurately classifying human emotions (joy, sadness, anger, fear, etc.) from raw text data, such as social media posts or reviews.

Key Achievements

  • Engineered a comprehensive text preprocessing pipeline using NLTK, including tokenization, lemmatization, and stop-word removal.
  • Trained and evaluated multiple models, identifying a Bidirectional LSTM as the top performer with a 94% test accuracy.
  • Utilized TF-IDF for baseline models (Logistic Regression, SVC) and Keras Tokenizer with padding for deep learning models.
  • Addressed class imbalance using computed class weights, significantly improving F1-scores for minority classes.

Tech Stack

Python, TensorFlow/Keras, Scikit-learn, NLTK, Pandas, Matplotlib, Jupyter.

Credit Approval Prediction Pipeline

Project Goal

To build a complete, reproducible machine learning pipeline that predicts credit approval status based on historical applicant data. A key challenge was addressing the severe class imbalance in the dataset.

Key Achievements

  • Conducted extensive EDA and advanced feature engineering, including log transformations and categorical encoding.
  • Successfully addressed the 8% approval rate class imbalance using the SMOTE (Synthetic Minority Over-sampling Technique).
  • Trained and optimized a LightGBM classifier, achieving a strong ROC-AUC score of ~0.85 on the holdout test set.
  • Generated comprehensive evaluation reports, including confusion matrices and feature importance plots.

Tech Stack

Python, LightGBM, Scikit-learn, Pandas, SMOTE, Matplotlib, Seaborn.

Sentiment Analysis of IMDB Reviews

Project Goal

To construct a sentiment classification pipeline to determine whether movie reviews are positive or negative, using a dataset of 50,000 IMDB reviews.

Key Achievements

  • Implemented a robust text preprocessing pipeline including tokenization, stemming, and stop-word removal.
  • Achieved 86% accuracy with a Random Forest model, representing a +15% improvement over the baseline.
  • Applied dimensionality reduction techniques (PCA, SVD) to enhance model performance and reduce overfitting.
  • Deployed the results in an interactive Python dashboard for real-time sentiment analysis.

Tech Stack

Python, Scikit-learn (Naive Bayes, SVM, Random Forest), PCA, SVD, Pandas.

Air Quality Forecasting & Analysis

Project Goal

To analyze over 100,000 air quality records from NYC and global datasets to forecast pollution levels and identify pollution hotspots.

Key Achievements

  • Developed a Random Forest model that achieved 85% forecasting accuracy with a Mean Squared Error of less than 0.02.
  • Created compelling geospatial pollution heatmaps using Plotly and Folium, successfully identifying hotspots across more than 50 locations.
  • Processed and cleaned large datasets containing NO2 and PM2.5 readings to prepare them for modeling.

Tech Stack

Python, Scikit-learn (Random Forest), Pandas, Plotly, Folium.
